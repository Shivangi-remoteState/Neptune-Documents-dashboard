# Neptune-Documents-dashboard
Frontend UI for a dashboard management system inspired by Neptune dashboard

This project is a responsive web dashboard interface for managing documents.
It contains a sidebar, user profile section, and a document listing area.

The project has been created as part of an academic assignment.

--------------------------------------------------

Features

* Sidebar with navigation links
* User profile card
* Document list with status labels
* Search input for documents
* Responsive layout for desktop and mobile
* Mobile friendly document view

--------------------------------------------------

Project Structure

/
index.html
style.css
README.md
Image files (icons and UI assets)

--------------------------------------------------

How to Run

1. Download or clone the project folder
2. Open the folder
3. Open index.html in any web browser

--------------------------------------------------

Technologies Used

HTML
CSS

--------------------------------------------------

Responsive Design

The layout automatically adjusts based on screen size.
On small screens the sidebar moves to the top and the document table changes to a stacked layout.

--------------------------------------------------

Assignment Information

This project was created for a frontend UI assignment to demonstrate layout design, CSS styling, and responsive web design.
